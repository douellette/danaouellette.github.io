What is Bootstrap?

  • Front End Framework
  • Basically just CSS and JS
  • Created by Twitter for consistency in internal projects
  • Supports responsive design grids
basically bootstrap is a layout, grid system, base css for tables, forms, buttons, icons, dropdowns, navbards, and more as well as a bunch of jquery plugins

Why Bootstrap?

  • Ease of use
  • Effeciency
  • Consistency
  • Complex interactive, responsive, and javascript features with only basic knowledge of html and css
